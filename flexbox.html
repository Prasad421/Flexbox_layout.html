ABOUT FLEX LAYOUT:
  1. Before the Flexbox Layout module, there were four layout.
  2. Block, for sections in a webpage.
  3. Inline, for text.
  4. Table, for two-dimensional table data.
  5. Positioned, for explicit position of an element.
  6. The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.
